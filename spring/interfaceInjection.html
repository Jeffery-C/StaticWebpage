Interface Injection (介面注入)

在mainApp中使用interface的class來建立物件

建構過程如下，首先建立要使用的interface class


package com.luv2code.springdemo;

public interface Coach {
    
    public String getDailyWorkout();
    
}

接著建立需要執行的多個class

1.BaseballCoach.java


package com.luv2code.springdemo;

public class BaseballCoach implements Coach {
    
    @Override
    public String getDailyWorkout() {
        return "Spend 30 minutes on batting practice";
    }

}

2.TrackCoach.java


package com.luv2code.springdemo;

public class TrackCoach implements Coach {
    
    @Override
    public String getDailyWorkout() {
        return "Run a hard 5k";
    }

}

最後建立我們的mainApp


package com.luv2code.springdemo;

public class MyApp {

    public static void main(String[] args) {

        // create the object
        Coach theCoach = new BaseballCoach();
        
        // use the object
        System.out.println(theCoach.getDailyWorkout());

        Coach theCoach = new TrackCoach();

        System.out.println(theCoach.getDailyWorkout());
    }

}

可以看到當需要更換interface class的物件Coach時，我們只需要更改右方物件名稱即可其餘地方不須更動，另外interface class底下的class必須有interface class的方法，因此不需擔心建立物件時沒有方法。
